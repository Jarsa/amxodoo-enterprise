Mexico - CFDI Payment Complement Follow-up

Track and validate CFDI payment complements for PPD invoices

This module automates the tracking, validation, and operational control of CFDI Payment Complements (Complementos de Pago PPD) for vendor payments and customer collections.

  • Automatic CFDI payment state on reconciled payments
  • Manual button to request complement from vendors
  • Automatic XML complement validation (8 rules)
  • Activity creation on validation errors
  • Scheduled follow-up reminders
  • Per-company start date configuration
